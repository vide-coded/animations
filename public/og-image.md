# Open Graph Image

This directory should contain the Open Graph image for social media sharing.

## Required Image:
- **Filename**: `og-image.png`
- **Dimensions**: 1200 × 630 pixels
- **Format**: PNG or JPG
- **File size**: < 1 MB

## Recommended Content:
- App logo/branding
- Tagline: "Canvas Animation Gallery"
- Visual preview of featured animations
- Dark background with colorful animation preview

## Quick Generation Options:

### Option 1: Use Figma
1. Create a 1200×630 frame
2. Add Brain-Rot branding
3. Export as PNG

### Option 2: Use Canva
1. Search for "Social Media" → "Facebook Post"
2. Resize to 1200×630
3. Design and export

### Option 3: Use code (React)
Create a Next.js OG image route or use Puppeteer to screenshot:
- Canvas with particle animation background
- Title overlay: "Brain-Rot Canvas Animation Gallery"
- Subtitle: "24 Mesmerizing Animations"
